This is a virtual Visit of the upper part of Saint Guilhem le Désert (South of France).

On Mobile device, you can switch the view mode to enter VR.

The photo capture is made on an iPhone 13 Pro with a modified version of the sample code 10076: Create 3D Models with Object Capture.

The model is made from the 19.000 pictures with Agisoft Metashape and post process on Blender and MeshLab.

The rendering engine is based on BabylonJS.
